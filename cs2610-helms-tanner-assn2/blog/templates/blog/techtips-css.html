< - This character entity is used to show the left side of the code

> - This character entity is used to show the right side of the code

3. CSS Selectors

type - The type CSS selector is used when you want to select certain html elements and use css to style it

.class - The class CSS selector is used when you want to create some styling for an element. After you create the styling for the elements you want, in order to use it put a class="(class name)" to attach the styling to the element

#id - To use the #id CSS selector, create the stying with the #(id of element) in your stylingsheet. Then to connect the styling to the HTMl element, in the element tag put id="(name of css style)"

:nth-child - This CSS selector is used when you want to only change the styling of nth element within a HTML Element. For example, if you have a table and want every other line to be a different color, you could use tr:nth-child(odd) to select all of the odd rows and change the background color.

4. CSS Properties

background-color - This sets the background for the gived element that this css is tagged too. For example, if you want your background color to be blue, you could use "background-color: blue" or "background-color: red". Click on to toggle the background color of this.

text-align - text align defines how text is aligned within the parent element. For example, if your making a title for your page and you want the text to be in the center, you could use "text-align: center. Another way to use text-align is if you want all your text to look like the same length on each line. You could use "text-align: justify" to make all your text the same length. Im acutally using text-align justify in the current next. Notice how each line is the same length. Click on to toggle this text-align to center.

Display - Display is used when your trying to modify how that element is display on the screen. For example, you could use "display: none" if you want to hide an element. Later, you could use javascript and change the display of the element to "display: block" to show the element. click on to change this elements display to none;
